# Narration Script Reference

> 나레이션 분량 가이드 + 톤/전달 스타일 + 자막 규칙 + 타입별 차이 + 품질 체크리스트

## Core Principles

| 원칙 | 설명 |
|------|------|
| 영상 종속 | 나레이션은 영상의 보조 — 화면이 보여주는 것을 반복하지 않는다 |
| 자막 겸용 | 나레이션 = 자막 원본. 화면에 표시 가능한 길이와 구조를 유지한다 |
| 감정 동기화 | 샷/씬의 감정 톤(mood)과 나레이션 톤이 반드시 일치한다 |
| 호흡 여백 | 모든 나레이션 사이에 자연스러운 쉼(pause)을 확보한다 |
| 구어체 우선 | 글말이 아닌 입말 — 소리 내어 읽었을 때 자연스러워야 한다 |

## Duration별 분량 가이드

### 한국어 기준

| Duration | 글자 수 | 문장 수 | 비고 |
|----------|---------|---------|------|
| 3s | 20~30자 | 1문장 | 짧은 임팩트 |
| 5s | 40~70자 | 1~2문장 | 기본 단위 |
| 8s | 80~120자 | 2~3문장 | 설명형 |
| 10s | 100~150자 | 2~4문장 | 최대 권장 |
| 15s+ | 150~220자 | 3~5문장 | 분할 권장 |

**환산 공식**: 한국어 약 4~5자/초 (자연스러운 속도 기준)

### 영어 기준

| Duration | 단어 수 | 문장 수 |
|----------|---------|---------|
| 3s | 8~12 words | 1 sentence |
| 5s | 15~25 words | 1~2 sentences |
| 8s | 28~40 words | 2~3 sentences |
| 10s | 35~50 words | 2~4 sentences |

**환산 공식**: 영어 약 3 words/sec (moderate pace)

### 침묵 구간

모든 나레이션이 채울 필요는 없다. 다음 경우 의도적 침묵(무음)을 사용한다:

| 상황 | 효과 |
|------|------|
| 감정 전환 직후 | 여운 부여 |
| 임팩트 비주얼 | 영상에 집중 |
| 씬 전환 지점 | 호흡 전환 |
| 클라이맥스 직전 | 긴장감 고조 |

침묵 구간은 `[무음]` 또는 `[silence]`로 표기한다.

## 톤 & 전달 스타일

| 스타일 | 특징 | 적합한 장면 | 예시 |
|--------|------|-------------|------|
| 차분한 서술 | 낮은 톤, 일정한 속도, 감정 절제 | 일상 묘사, 회상 | "그날도 평범한 아침이었다" |
| 감성 내레이션 | 부드러운 톤, 감정 실림, 약간의 떨림 | 감동, 이별, 재회 | "그제서야 알았다. 그것이 마지막이었다는 걸" |
| 긴장/서스펜스 | 빠른 호흡, 짧은 문장, 점층적 고조 | 위기, 반전 | "문이 열렸다. 아무도 없었다. 아니, 있었다" |
| 밝은/경쾌 | 높은 톤, 리듬감, 활기 | 유쾌한 장면, 소개 | "자, 여기가 바로 그 유명한 곳입니다!" |
| 독백/고백 | 1인칭, 속마음, 내밀한 톤 | 캐릭터 심리, 갈등 | "솔직히 말하면, 나는 두려웠다" |
| 정보 전달 | 명확한 발음, 중립 톤, 키워드 강조 | 설명, 가이드, 정보 | "이 기술은 2024년 처음 도입되었습니다" |

## 자막 호환 규칙

나레이션은 영상 자막의 원본이 된다. 자막 표시를 고려한 규칙:

| 규칙 | 설명 |
|------|------|
| 한 줄 길이 | 한국어 15~20자, 영어 35~42자 이내 |
| 줄바꿈 | 의미 단위로 끊기 (조사/전치사 앞에서 끊지 않음) |
| 괄호/기호 | 최소화 — 자막에서 읽기 어려움 |
| 숫자 | 아라비아 숫자 사용 (자막 공간 절약) |
| 고유명사 | 첫 등장 시 풀네임, 이후 약칭 |
| 강조 | **굵게** 또는 *기울임* 대신 문맥으로 전달 (자막은 서식 제한) |

### 줄바꿈 예시

```
❌ "그는 어머니가 만들어 주
   신 도시락을 먹었다"

✅ "그는 어머니가 만들어 주신
   도시락을 먹었다"
```

## 콘텐츠 타입별 차이

### film (Shot 단위)

| 항목 | 규칙 |
|------|------|
| 단위 | 각 Shot의 `#### 나레이션 스크립트` 블록 |
| 분량 | Shot Duration 기준 (보통 5~8초) |
| 대사 우선 | 캐릭터 대사가 있는 샷은 대사 우선, 나레이션은 보조 또는 생략 |
| 캐릭터 연계 | Character Document의 성격/말투를 나레이션 톤에 반영 |
| 사운드 동기화 | BGM/SFX 타이밍을 고려하여 나레이션 시작/종료 조절 |
| 다중 화자 | 나레이터 + 캐릭터 대사가 겹치지 않도록 배치 |

### narration-video (Scene 단위)

| 항목 | 규칙 |
|------|------|
| 단위 | 각 Scene의 `### 나레이션 스크립트` 블록 |
| 분량 | Scene Duration 기준 (보통 10~20초) |
| 단일 화자 | 나레이터 한 명이 전체를 이끔 |
| 이미지 보완 | 정지 이미지 위에 나레이션이 주도 — 영상보다 스크립트 비중이 높음 |
| 감정 곡선 | 씬 간 감정 흐름이 자연스럽게 이어지도록 전환 설계 |
| 자막 = 나레이션 | 나레이션 전문이 그대로 자막으로 표시됨 |

## Weak vs Strong

| 요소 | 약한 표현 | 강한 표현 |
|------|-----------|-----------|
| 도입 | "이것은 한 남자의 이야기입니다" | "그 남자는 매일 같은 길을 걸었다" |
| 감정 | "그녀는 슬펐다" | "눈물이 마르기도 전에, 또 다른 전화가 울렸다" |
| 전환 | "다음 장면입니다" | "그로부터 3년 뒤—" |
| 설명 | "이 마을은 아름답습니다" | "골목마다 빛바랜 간판이, 세월을 대신 말해주고 있었다" |
| 독백 | "나는 후회했다" | "돌아갈 수 있다면, 그날 그 말만은 하지 않았을 텐데" |
| 긴장 | "무서운 일이 일어났다" | "소리가 멈췄다. 갑자기, 모든 것이" |
| 정보 | "AI는 중요합니다" | "2025년, AI는 우리 삶의 73%를 바꿔놓았습니다" |

### Show, Don't Tell

나레이션의 핵심 원칙: **설명하지 말고 보여줘라**

```
❌ Tell: "할아버지는 손녀를 매우 사랑했다"
✅ Show: "할아버지는 매일 아침, 손녀의 신발을 가지런히 놓아두었다"

❌ Tell: "그날 비가 많이 왔다"
✅ Show: "우산 없이 걷는 사람들 사이로, 빗물이 골목을 채웠다"
```

## 감정 곡선 설계

전체 영상의 나레이션은 감정 곡선(Emotional Arc)을 따른다:

| 구간 | 감정 흐름 | 나레이션 특징 |
|------|-----------|--------------|
| 도입 (1~2씬) | 안정/호기심 | 차분한 서술, 세계관 소개 |
| 전개 (3~5씬) | 몰입/공감 | 구체적 에피소드, 감정 이입 |
| 위기 (6~8씬) | 긴장/갈등 | 짧은 문장, 빠른 호흡, 침묵 활용 |
| 클라이맥스 (9~10씬) | 최고조 | 감정 폭발 또는 반전, 핵심 메시지 |
| 해소 (11~12씬) | 여운/성찰 | 느린 속도, 여백, 마무리 |

**주의**: 감정이 한 방향으로만 가면 단조로워진다. 전개 중간에 작은 반전(유머, 의외의 장면)을 넣어 리듬을 만든다.

## Pre-Writing Checklist

- [ ] 전체 영상의 감정 곡선이 설계되었는가
- [ ] 각 씬/샷의 Duration이 확정되었는가
- [ ] 나레이션 톤/전달 스타일이 결정되었는가
- [ ] 화자(나레이터)의 페르소나가 정의되었는가
- [ ] 대사가 있는 샷의 나레이션/대사 우선순위가 정해졌는가
- [ ] 의도적 침묵 구간이 계획되었는가

## Post-Writing Checklist

- [ ] 모든 나레이션이 Duration 기준 분량 범위 내인가 (±20%)
- [ ] 소리 내어 읽었을 때 자연스러운가 (구어체)
- [ ] 씬/샷의 감정 톤과 나레이션 톤이 일치하는가
- [ ] 자막 한 줄 길이가 15~20자(한) / 35~42자(영) 이내인가
- [ ] 전후 씬과의 감정 전환이 자연스러운가
- [ ] 화면이 보여주는 것을 나레이션이 반복하지 않는가 (Show, Don't Tell)
- [ ] 침묵 구간이 적절히 배치되었는가
- [ ] 경어체/한다체가 전체에서 일관되는가

## Notes

- 나레이션 분량은 Duration의 70~90%를 채우는 것이 적절 — 100% 채우면 숨 막힌다
- film에서 대사와 나레이션은 절대 겹치지 않는다
- narration-video에서 나레이션이 유일한 음성 요소이므로 스크립트 품질이 영상 품질을 결정한다
- 번역/다국어 대응 시 분량이 달라지므로 Duration 여유를 둔다
